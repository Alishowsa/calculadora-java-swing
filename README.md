# Calculadora Java Swing

Projeto desenvolvido em Java utilizando a biblioteca Swing para criação de uma interface gráfica desktop.

## Funcionalidades

* Soma
* Subtração
* Multiplicação
* Divisão
* Interface gráfica amigável

## Tecnologias Utilizadas

* Java 25
* Swing
* Git
* GitHub
* VS Code

## Estrutura do Projeto

```
src/
├── Main.java
└── CalculadoraSimplesAlisson.java
```

## Como Executar

### Compilar

```bash
javac src/*.java
```

### Executar

```bash
java -cp src Main
```
